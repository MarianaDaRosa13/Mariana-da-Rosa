# Mariana da Rosa — academic website

Personal academic website for Mariana da Rosa, hosted with GitHub Pages.

## Updating the content

- Main content: `index.html`
- Colors and layout: `assets/css/style.css`
- Profile photo: replace the GitHub avatar URL in `index.html`
- CV: add the PDF to an `assets/files/` folder and update the CV link

The site is a static HTML/CSS project and does not require a build step.
